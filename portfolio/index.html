Equipment rental

Marvel information portal

This site was created during the React course. It uses APIs sent from the Marvel website and processes them to display information. All content is based on it.

Online store

The store was built from scratch on React using Firebase technologies. Implemented: an administrative panel for managing site content and editing the existing one, an order system, a user account with email confirmation. The site is fully responsive and uses its own Hook to interact with Firebase, as well as Context to transfer content between components.

Order

I have implemented the order process, which, after placing an order, appear based on the specified email in the user's account.

Order system

When the user places an order, it appears in the admin panel, where you can perform various actions: change the status, add a consignment note number, change information, delete. And according to the changes, all of them are displayed in the user's account.

Personal cabinet

The user can register via email, which will allow him to track orders, as well as fill in the data that will then be filled in automatically when ordering.

Wish list and order cart system

The product that the user has liked or added to the cart is stored in the local storage as an id. This allows you not to lose it when you update or re-enter the store. You can also leave a review if you are registered.

Manage categories

The admin panel allows you to create, delete, and edit categories and products.

Manage goods

When you create a product, you create a gallery, description, properties (which are then pulled into filters), various flags (promotion, out of stock, new product, top product)... And of course, all this can be edited.
